## 2. 서버 실행 방법

### 필요한 것
- 관리자 노트북 (Node.js 설치 필요 → nodejs.org)
- 같은 와이파이 또는 인터넷 연결

### Render.com 클라우드 배포 (추천)
```
1. render.com 접속 → 로그인
2. 기존 서비스 클릭 → Manual Deploy
3. URL 공유: https://kfcccmg-quize.onrender.com
⚠️ 15분 미사용 시 절전모드 → 교육 시작 10분 전 미리 접속해 깨워두세요!
```

### 로컬 실행 (같은 와이파이 환경)
```bash
cd kmg-quiz
npm install   # 최초 1회만
npm start
# 터미널에 표시된 http://192.168.x.x:3000 주소를 참가자에게 공유
```

## 3. 참가자 사용법

### 🔵 Step 1: 앱 접속
- 브라우저에서 제공받은 URL 접속
- (예: https://kfcccmg-quize.onrender.com)

### 🔵 Step 2: 등록
1. **이름** 입력 (실명)
2. **팀 번호** 드롭다운에서 내 팀 선택 (1팀~14팀)
3. **등록하고 입장** 버튼 클릭
4. → 화면에 **내 대결 번호** (예: 7번)가 크게 표시됨
5. 이 번호를 **상대방에게 알려주세요**

### 🔵 Step 3: 대결 신청
1. 직접 만난 상대방과 **번호를 교환**
2. 로비에서 **⚔️ 대결 신청하기** 버튼 클릭
3. 상대방 번호 입력 → **도전!**
4. 상대방이 수락하면 → **문제 유형 랜덤 공개** → 대결 시작!

### 🔵 Step 4: 대결 진행
- 문제가 화면에 표시됨
- **6초 이내**에 답을 선택
- **먼저 맞추면 승점** 획득
- 5문제 중 3점 먼저 획득한 사람이 승리!

### 🔵 Step 5: 결과 확인
- 대결 결과 화면에서 승패 확인
- **전체 순위 보기**로 현재 순위 확인

### ⚠️ 참가자 규칙
- 한 번 대결한 상대는 다시 대결 불가
- 재접속 필요 시: 시작 화면 → **재접속** → 내 번호 입력

## 4. 관리자 사용법

### 관리자 로그인
- 시작 화면 하단 **관리자 입장** 섹션
- 비밀번호: **`2026`**
- 로그인 후 **PC 대시보드** 화면으로 이동

### 관리자 탭 구성

#### 📊 실시간 현황 탭
- 참가자 수, 접속 중, 대결 중 등 KPI 6개 실시간 표시
- 개인 TOP 5 / 팀 TOP 5 자동 업데이트
- 최근 대결 결과 카드 자동 갱신
- **새로고침 불필요 — 자동 실시간 업데이트**

#### 🏆 개인 순위 탭
- 전체 개인 순위 (승률/정답/대결 수 기준 정렬 가능)
- 승률 막대 그래프, 접속 상태 표시
- PC 화면에 띄워서 참가자들과 공유하기 좋음

#### 🏅 팀 순위 탭
- 팀 평균 승률 / 정답 합계 / 대결 합계 — 3개 기준 동시 비교
- 팀별 구성원 상세 (이름, 승률, 정답 수)
- PC 화면 투사용으로 최적화

#### ⚔️ 대결 피드 탭
- 대결이 끝날 때마다 실시간 자동 추가
- 현재 대결 중인 선수 목록
- 오늘의 통계 (완료 대결, 누적 정답, 평균 승률)

#### 📋 문제 관리 탭
- 문제 목록 확인 (OX/3지/4지 탭 전환)
- 문제 추가 / 수정 / 삭제
- 엑셀 파일 업로드로 한꺼번에 교체

#### ⚙️ 제어 탭
- 🔥 **막판뒤집기 ON/OFF** (가장 중요!)
- 수동 패널티 적용 (거부자 이름 입력)
- 점수 초기화 / 전체 초기화

## 5. 막판뒤집기 스테이지

### 개념
교육 세션 마지막에 **역전 기회**를 주는 특별 스테이지입니다.  
이 모드에서는 모든 승리가 **×3점**으로 계산됩니다.

### 관리자 설정 방법
1. 관리자 대시보드 → **⚙️ 제어** 탭
2. 🔥 **막판뒤집기 ON** 버튼 클릭
3. → 모든 참가자 화면에 빨간 배너 자동 표시
4. → 전체 공지 팝업도 자동 발송

### 막판뒤집기 규칙

| 상황 | 결과 |
|------|------|
| 대결 승리 | **+3점** (일반 모드의 3배) |
| 대결 패배 | -3점 |
| 대결 **거부** | 즉시 패배 + **보유 승수 ×3 차감** |

> 예: 승수 10점 보유 중 거부 시 → 10×3=30점 차감 → 승수 0점 + 패배 3추가

### 거부 패널티 처리

#### 자동 처리 (앱에서)
- 상대방이 도전 신청 시 **거절** 버튼 클릭 → 자동으로 패널티 적용

#### 수동 처리 (관리자)
- 직접 거부했거나 앱 밖에서 거부한 경우
- ⚙️ 제어 탭 → **수동 패널티 적용**
- 거부자 이름 입력 → **패널티 적용** 버튼
- → 해당 선수 승수 ×3 차감 자동 적용

### 종료 방법
- ⚙️ 제어 탭 → **⏹ 종료 / OFF** 버튼

## 6. 수상 기준

### 🏅 팀상 (3개 부문)

| 수상 | 기준 | 확인 방법 |
|------|------|-----------|
| 팀 승률왕 🥇 | 팀원 평균 승률 1위 | 팀 순위 → 평균승률 탭 |
| 팀 정답왕 🎯 | 팀원 정답 합계 1위 | 팀 순위 → 정답합계 탭 |
| 팀 열정왕 🔥 | 팀원 대결 수 합계 1위 | 팀 순위 → 대결합계 탭 |

### 🏆 개인상 (3개 부문)

| 수상 | 기준 | 확인 방법 |
|------|------|-----------|
| 개인 승률왕 🥇 | 승률 1위 | 개인 순위 → 승률순 |
| 개인 정답왕 🎯 | 정답 수 1위 | 개인 순위 → 정답순 |
| 개인 대결왕 🔥 | 대결 수 1위 | 개인 순위 → 대결순 |

### 시상 진행 방법
1. 관리자 대시보드 → **🏅 팀 순위** 탭 화면 투사
2. **🏆 개인 순위** 탭 화면 투사
3. 각 기준별 1위 발표

## 7. 문제 추가·수정 방법

### 방법 A: 앱에서 직접 수정 (간편)
1. 관리자 로그인 → **📋 문제 관리** 탭
2. 섹터 선택 (OX / 3지 / 4지)
3. **✏️ 수정** 또는 **🗑 삭제** 버튼 클릭
4. **+ 문제 추가** 버튼으로 새 문제 추가

### 방법 B: 엑셀로 일괄 업로드 (권장)
1. 제공된 **`새마을금고_퀴즈문제_편집용.xlsx`** 파일 열기
2. 해당 시트에서 문제 수정/추가/삭제
3. **저장** (Ctrl+S)
4. 관리자 → **📋 문제 관리** 탭 → **📤 엑셀 파일 선택**
5. 업로드 → 즉시 적용!

### 엑셀 작성 규칙

| 섹터 | 정답 형식 |
|------|-----------|
| OX 퀴즈 | 반드시 대문자 **O** 또는 **X** |
| 3지선다 | 보기 3개 중 하나와 **정확히 동일한 텍스트** |
| 4지선다 | 보기 4개 중 하나와 **정확히 동일한 텍스트** |

> ⚠️ 정답과 보기의 텍스트가 한 글자라도 다르면 아무도 맞출 수 없습니다!

## 8. 자주 묻는 질문 FAQ

**Q. 앱에 접속이 안 돼요.**  
A. 서버가 절전모드일 수 있습니다. 브라우저에서 URL을 입력하고 30~50초 기다리면 깨어납니다. 이후 정상 작동합니다.

**Q. 등록했는데 번호를 잊어버렸어요.**  
A. 시작 화면 → **재접속** → 관리자에게 번호 확인 요청 (관리자 대시보드 개인 순위에서 이름으로 찾을 수 있음)

**Q. 대결 신청했는데 상대방 화면에 안 나타나요.**  
A. 상대방이 앱을 새로고침하거나 재접속하면 해결됩니다.

**Q. 이미 대결한 상대와 또 대결하고 싶어요.**  
A. 새마을금고법상 각 사람과 1회만 대결 가능합니다. 더 많은 사람을 찾아 대결하세요!

**Q. 섹터(문제 유형)를 미리 알 수 있나요?**  
A. 불가능합니다. 대결 수락 순간 서버에서 랜덤으로 결정됩니다.

**Q. 막판뒤집기 중 실수로 거부 버튼을 눌렀어요.**  
A. 관리자에게 즉시 알려주세요. 관리자가 수동으로 패널티를 되돌릴 수 있습니다 (점수 수정).

**Q. GitHub/Render 재배포가 필요한가요?**  
A. 문제 수정은 앱 내에서 바로 가능합니다. 코드 자체를 수정할 때만 GitHub → Render 배포가 필요합니다.

**Q. 관리자 비밀번호를 바꾸고 싶어요.**  
A. `server.js` 파일의 `const ADMIN_PASSWORD = '2026';` 부분을 수정하고 재배포하면 됩니다.
